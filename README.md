## Stack

- [Astro 6](https://astro.build/) вЂ” content-first SSG, zero JS by default
- [React 19](https://react.dev/) вЂ” interactive islands via `@astrojs/react`
- [Tailwind CSS v4](https://tailwindcss.com/) вЂ” via `@tailwindcss/vite`
- [shadcn/ui](https://ui.shadcn.com/) вЂ” style `radix-nova`, base `neutral`, icons `lucide`
- [motion](https://motion.dev/) вЂ” animation library (CSS-only reveals in v1; motion is installed for richer interactions)
- [Lenis](https://lenis.dev/) вЂ” smooth scroll (desktop only, off on touch + reduced-motion)
- [Fraunces](https://fonts.google.com/specimen/Fraunces) + [Inter](https://fonts.google.com/specimen/Inter) вЂ” display serif + body sans, both with Greek subsets, via Astro's native Fonts API

## Commands

| Command           | What it does                                    |
| ----------------- | ----------------------------------------------- |

| `npm run dev`     | Start the dev server at <http://localhost:4321> |
