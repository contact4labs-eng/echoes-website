# Echoes Bistrot

Premium one-page marketing website for **Echoes Bistrot** — brunch / bistro at Νυμφών 33, Ηλιούπολη 163 41 (Athens). Bilingual Greek + English, SSG, deploys to Vercel.

> **Managing this project?** Read **[MANAGER_BRIEFING.md](./MANAGER_BRIEFING.md)** first — it's the canonical context document covering brand facts, stack decisions, current state, open TODOs, and the manager ↔ developer collaboration protocol.

## Stack

- [Astro 6](https://astro.build/) — content-first SSG, zero JS by default
- [React 19](https://react.dev/) — interactive islands via `@astrojs/react`
- [Tailwind CSS v4](https://tailwindcss.com/) — via `@tailwindcss/vite`
- [shadcn/ui](https://ui.shadcn.com/) — `radix-nova` preset (drawer primitive only)
- [motion](https://motion.dev/) — for the one scroll-driven cinematic moment + small interactive hooks
- [Lenis](https://lenis.dev/) — smooth scroll (desktop only, off on touch + reduced-motion)
- [Fraunces](https://fonts.google.com/specimen/Fraunces) + [Inter](https://fonts.google.com/specimen/Inter) — display serif + body sans, both with Greek subsets, via Astro's native Fonts API

## Homepage — the 8 movements (R3 "Slow Hours")

The homepage is a vertical film, not a brochure. Each movement is a deliberate section with its own visual register:

1. **Threshold** — full-bleed bougainvillea facade. Identity strip at the bottom. Subtle cursor parallax. Header stays usable from second one.
2. **First Word** — near-empty cream panel with a single quiet sentence. Establishes voice.
3. **Through the Door** — the one scroll-driven cinematic moment. Two photos crossfade (exterior → interior) as the user scrolls a ~150svh pinned section.
4. **Slow Hours** — editorial daypart triptych. Optional time-of-day highlight that degrades gracefully to a static triptych if JS is off.
5. **From the Kitchen** (`#menu`) — three full-width editorial dish panels, alternating image-left/image-right. No card grid.
6. **The Pass** (`#kitchen`) — asymmetric photo essay of the kitchen on dark navy. No personal chef name. No `01 / 02 / 03` row. One short body line + the `Coffee by Taf` credit.
7. **Nymfon 33** (`#visit`) — visit info treated as a tilted physical postcard. Address as typography, hours table, primary phone CTA. Maps + Instagram as small side links.
8. **End Card** — full-bleed dessert photo with a quiet phrase (`περάστε από τη Νυμφών.` / `come by Nymfon.`) and a tiny `Call · Maps · Instagram` action row.

The trust signals (noupou.gr press citation, Taf Coffee partnership) live as a single quiet line in the footer credit and as a caption in The Pass. **No dedicated Social Proof section, no cards.**

## Project structure

```
src/
  assets/photos/         Editorial photos (auto-optimized to WebP by Astro). Menu thumbs in /menu.
  components/
    layout/              Header, Footer, LangSwitcher, MobileNav (the only React island in chrome)
    sections/            Threshold, FirstWord, ThroughTheDoor, SlowHours, FromTheKitchen,
                         ThePass, Nymfon33, EndCard
    interactive/         CrossfadeOnScroll (Through-the-Door), SlowHoursActive (time highlight)
    ui/                  shadcn primitives (button, card, sheet)
    HomeContent.astro    Composes the 8 movements
  content/menu.ts        3 dishes for From the Kitchen (slug + image; names via i18n)
  hooks/                 useScrollProgress, useCursorParallax (motion + reduced-motion guards)
  i18n/                  el.json, en.json, utils.ts (getLang, useTranslations, getLocalePath, getAlternates)
  layouts/Layout.astro   Full SEO <head> + Lenis init + IntersectionObserver for reveals + rAF
                         header scroll state + active-nav scroll spy
  lib/
    config.ts            Site singleton (phone, address, social; geo + hours = null pending client)
    schema.ts            buildRestaurantSchema(locale) — emits geo/openingHoursSpec only when confirmed
    utils.ts             cn() helper
  pages/
    index.astro          Greek home
    en/index.astro       English home
  styles/
    global.css           Brand tokens (OKLCH) + shadcn mappings — no dark mode
    motion.css           Reveal classes + Ken Burns + header dual-state vars + reduced-motion guards
public/
  og-image.jpg, robots.txt, favicon.svg
```
